使用说明

  • 本页管理公众移动云电脑 Path B 保活（非爱家线）。
  • 上方登录；短信区常显（设备信任/增强策略授权）。
  • 卡片「启动」= Path B 桌面保活（账号登录态随 CLI 自动开启）；「停止」同步停保活。
  • 删除账号在「配置」弹窗内（需二次确认）。
  • 关掉浏览器后，已启动的保活仍在服务端后台运行。
  • 可选「访问密钥」：首次可设置，之后顶栏可改/关；请求带 Bearer。
